# Personal Website — Umberto Di Laudo

A minimal, responsive personal portfolio website built with plain HTML, CSS, and JavaScript. No frameworks, no build tools.

## Features

- Dark / light theme toggle (saved in `localStorage`)
- Animated particle background on the hero section
- Typing effect for rotating roles
- Scroll-triggered animations
- Responsive design with mobile hamburger menu
- Sections: About, Education (timeline), Skills, Publications & Projects, Contact

## Tech Stack

- **HTML / CSS / JavaScript** — no frameworks
- **[Lucide Icons](https://lucide.dev/)** — via CDN (pinned version)
- **[Google Fonts](https://fonts.google.com/)** — Inter + Playfair Display
- **Cloudflare** — hosting, CDN & DNS

## Project Structure

```
.
├── index.html      # Main page
├── style.css       # All styles (light & dark theme)
├── script.js       # Particles, typing effect, theme toggle, animations
├── photo/          # Profile photo
└── CV/             # PDF curriculum vitae
```

## Local Development

Just open `index.html` in a browser — no server or build step required.
